# Savdo Mobil Dasturi

Professional mobil dastur React Native (Expo) asosida yaratilgan.

## Xususiyatlar

- ✅ Online va Offline rejimda ishlaydi
- ✅ GPS tracking - ish vaqtida avtomatik joylashuv yuborish
- ✅ Buyurtmalar yaratish va boshqarish
- ✅ Mahsulotlar va mijozlar ro'yxati
- ✅ Offline data sync
- ✅ Professional UI/UX
- ✅ Push notifications

## Dasturni ishga tushirish

```bash
cd mobile_app
npm install
npx expo start
```

## Build qilish

```bash
# Android
npx expo build:android

# iOS
npx expo build:ios
```

## Texnologiyalar

- React Native (Expo)
- SQLite (Offline database)
- React Navigation
- AsyncStorage
- React Query (Data fetching)
- Expo Location (GPS tracking)
- Expo Background Fetch
- React Context (State management)
